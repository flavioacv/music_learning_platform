# CONTENT_GENERATION_RULES.md

# Regras para Geração de Conteúdo

## Objetivo

Todo conteúdo deve priorizar prática antes de teoria.

---

# Linguagem

Utilizar linguagem simples.

Evitar termos acadêmicos.

Evitar jargões desnecessários.

Explicar como para alguém sem experiência musical.

---

# Estrutura Obrigatória

Toda aula deve conter:

1. Objetivo
2. Conceito
3. Analogia
4. Exemplo Visual
5. Exercício Guiado
6. Exercício Prático
7. Desafio
8. Resumo
9. Quiz

---

# Regras Pedagógicas

Primeiro mostrar.

Depois explicar.

Depois praticar.

Nunca inverter essa ordem.

---

# Exercícios

Toda aula deve possuir:

* Mínimo 5 exercícios guiados
* Mínimo 10 exercícios práticos
* 1 desafio final

---

# Dificuldade

A dificuldade deve aumentar gradualmente.

Cada aula só pode introduzir um novo conceito principal.

---

# Gamificação

Conceito concluído:
+10 XP

Exercício correto:
+20 XP

Desafio concluído:
+50 XP

Quiz aprovado:
+100 XP

---

# Retenção

Ao final de cada aula a IA deve responder:

"O aluno está mais próximo de tocar uma música?"

Se a resposta for não, a aula deve ser revisada.
